Deployment Guide
Go Live in Minutes
Deploy the Streamlit backend or use the standalone HTML frontend — zero server required for the client-side version.
01
1
Train the Model

Run the training script to generate loan_model.pkl. Gradient Boosting trains on 2,000 records in under 30 seconds.

python train_model.py
02
2
Push to GitHub

Commit all files including loan_model.pkl — Streamlit Cloud needs the .pkl to serve predictions. Do NOT add it to .gitignore.

git add . && git push origin main
03
3
Deploy to Streamlit Cloud

Go to share.streamlit.io → New App → select repo → file: app.py → Deploy. Your dashboard goes live in ~2 minutes, for free.

share.streamlit.io → New App → app.py
Deploy HTML frontend — zero backend needed
